AI 探索

没有技术背景。最开始用 ChatGPT 是靠绕过封锁才注册上的。现在在用 Claude Code 做两年前根本想不到自己能做的东西。

2022–
2023

怎么开始的

翻过那道墙

第一次知道 ChatGPT，注册就是个问题——IP 封锁，账号限制，国内的那套全碰上了。靠计算机系的同学帮忙，各种办法绕过去的。

注册上之后用得也简单，主要就是当升级版的搜索引擎。帮忙写过论文，回答问题，速度比百度快。有用，但也没觉得有多特别。

ChatGPT

2025
年底

认真起来了

NotebookLM 让我觉得不一样了

迪卡侬实习那段时间，意外白嫖了一年 Gemini Pro。开始用在工作上——起草文件、总结资料、中英文快速互译。

然后发现了 NotebookLM。逻辑很直接：你把自己的资料喂进去——PDF、文档、YouTube 链接都行——它只在你给的范围内作答，不会乱引用互联网上的东西。拿来做研究、写报告、整理信息，一下子觉得特别好用。

那是第一次我觉得 AI 是一个系统，而不只是一个功能。

学怎么沟通

训练 Prompt

去悉尼读研之后开始更刻意地用 ChatGPT——不只是要输出结果，而是去搞懂怎么问才有用。好的 Prompt 是什么样的，加什么上下文会让结果变好，怎么从一个模糊的想法走到真正有用的东西。

这部分大多数人会跳过。我觉得值得花时间。

发现 Claude

从聊天工具到别的什么

之前就听说 Claude Code 在开发者里很火，但觉得跟我没关系——我不写代码。最开始用 Claude 也就是普通的聊天助手，跟其他的没什么区别。

然后开始摸 Projects——有持久上下文的长期对话。再到 Claude Code，一直在回避，后来还是用了。用了之后发现我之前担心的那些（技术门槛太高），其实根本不是我以为的那道坎。

Vibe Coding

我居然做了个产品

第一个项目：个人记账工具。本来想做 App，被 Claude 劝成了网页版——技术栈更简单，直接部署，手机电脑都能用。

买了阿里云域名，部署上去了。界面很糙，我完全不在意。

在迪卡侬的时候，我近距离见过完整的产品研发流程：BRD、PRD、开发迭代、测试、UAT、各方签字。一整个团队，好几个月。这次我一个人从想法到上线，一个周末搞完，没有那些正式流程，但完整。

就是那一刻，我对自己能做什么的认知变了。

Vibe Coding 阿里云

不懂代码，大概也不会专门去学。但「我没法自己做东西」这个念头，已经不在了。

现在

接下来

个人网站、知识库，以及再往后

这个网站是第二个作品。用 Claude Code，同样的方式——描述想要什么，不断迭代，部署上线。比预期慢，但做出来的是真的东西。

我也在用 Obsidian 搭个人知识库——把多年来那些没有被好好留下来的想法和观察放进去。我有不少觉得值得记的想法，后来都消散了。这个习惯我不想继续了。

我没有一个现在就想做出来的具体产品。我有一个方向：把这些工具理解得足够深，等到我真的有一个值得去做的想法的时候，技术这边不是拦住它的理由。
